### 6. URL Structure & Permalinks

#### A. Clean URLs with mod_rewrite
```php
// .htaccess configuration
RewriteEngine On
RewriteBase /

# Remove .php extension
RewriteCond %{REQUEST_FILENAME} !-d
RewriteCond %{REQUEST_FILENAME}\.php -f
RewriteRule ^(.*)$ $1.php [L]

# SEO-friendly URLs
RewriteRule ^constituency/?$ index.php [L]
RewriteRule ^candidate/([a-z-]+)/?$ candidate.php?slug=$1 [L]
RewriteRule ^blog/([a-z0-9-]+)/?$ blogs.php?post=$1 [L]
```

#### B. Breadcrumb Navigation
```php
<?php
function generate_breadcrumbs($current_page) {
    $breadcrumbs = [
        'Home' => '/',
        'Constituency Data' => '/constituency',
        'Candidate Profile' => '/candidate',
        'News & Updates' => '/blogs'
    ];
    
    // Schema.org BreadcrumbList
    $schema = [
        "@context" => "https://schema.org",
        "@type" => "BreadcrumbList",
        "itemListElement" => []
    ];
    
    $position = 1;
    foreach ($breadcrumbs as $name => $url) {
        $schema['itemListElement'][] = [
            "@type" => "ListItem",
            "position" => $position++,
            "name" => $name,
            "item" => "https://yourdomain.com" . $url
        ];
        if ($name === $current_page) break;
    }
    
    return json_encode($schema);
}
?>
```
